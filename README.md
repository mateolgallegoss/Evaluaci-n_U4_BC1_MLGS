# Evaluación_U4_BC1_MLGS
## Proceso de configuración

1. Una vez instalado Github Desktop y Visual Studio Code,
![Paso 1](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%201.png?raw=true "Paso 1")

2. Se presiona Ctrl+N en Github Desktop, o se accede a File > New Repository
![Paso 2](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%202.png?raw=true "Paso 2")

3. En la ventana emergente, se proceden a llenar los siguientes datos:
    1. Name: Evaluación_U4_BC1_(Iniciales Nombre)
    2. Local path: se selecciona un directorio para guardar el área de trabajo, stage y repositorio.
    3. Enmarcar la casilla para inicializar el repositorio con un README
    4. La licencia y la descripción son opcionales, pero brindan mayor profesionalidad al repositorio. 
![Paso 3](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%203.png?raw=true "Paso 3")

4. Seleccionar la opción Publish repository, para cargar el nuevo directorio dentro de Github.
![Paso 4](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%204.png?raw=true "Paso 4")

5. Se presiona Ctrl+Shift+A para abrir el directorio en el editor externo configurado, en este caso, Visual Studio Code.
![Paso 5](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%205.png?raw=true "Paso 5")

6. Se crea un archivo en formato markdown con el nombre "TEST".
![Paso 6](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%206.png?raw=true "Paso 6")

7. Ahora, puede proceder a editar el README y el TEST con las indicaciones otorgadas en Microsoft Teams (No modificar .gitattributes sin conocimiento sobre git)
![Paso 7](https://github.com/mateolgallegoss/Evaluaci-n_U4_BC1_MLGS/blob/main/Imagenes/Paso%207.png?raw=true "Paso 7")
